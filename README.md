# AngularJS demo application integrated with dummy apis(mocky.io).

# Dependencies
Node: 13.1.0
Angular: 8.2.14

## Installation Steps
### Clone the repo
    git clone https://github.com/thapa26/CompanyProject.git
### Switch to directory
    cd <DIR_NAME>
### Install dependencies
    npm i
    npm install chart.js --save
### Start Application
    ng serve

Open the following link the browser
 http://localhost:4200/
