# AngularJS demo application integrated with dummy apis(mocky.io).

## Dependencies
Node: 13.1.0
Angular: 8.2.14

## Installation Steps
1. Clone the repo
    git clone <LINK>
2. Switch to directory
    cd <DIR_NAME>
3. Install dependencies
    npm i
4. Start Application
    ng serve

Open the following link the browser
 http://localhost:4200/
